API Endpoint Documentation

GET /api/printers

Retrieves a list of all discovered printers, including their status, connection type, OS name, and other details.

POST /api/print

Sends a print job to a specified printer using a defined template and data. Requires a JSON body with `printerName`, `templateType`, and `templateData`.
